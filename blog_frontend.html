Тема: Изучение HTML

Изучил теги для адaптивных изображений.

Тег <img> с парметрами srcset и sizes выбирает изображения исходя из разрешения view port сайта на экране устройства. Если нет подходящего, то берется изображение из параметра src. 

          
            <img
            srcset="file_name1 001w, file_name2 002w, ..."
            sizes="(max-widht/min-width: 001px) 001px,
            (max-widht/min-width: 002px) 002px,
            ..., 00npx"
            src="file_name"
            alt="">

Тег <picture> - вместо <img> умеет выбирать изображение исходя из разрешения view port сайта на экране устройства.

          
            <picture>
            <source srcset="file_name1" media="(max-widht/min-width: 001px)">
            <source srcset="file_name2" media="(max-widht/min-width: 002px)">
            ...
            <source srcset="file_name" media="(max-widht/min-width: 00npx)">
            <img src="file_name" alt="">
            </picture>
